# One-Extra-Plate (Food Donation App)
One extra plate  has been concerting all its efforts towards providing fresh and nutritious meals to the poor and needy  on every single day across different locations . We are continuously leveraging technology to multiply our reach. Our website focuses on , allowing the users to create an account through which they can reach out to the volunteers who can also create an account and connect and interact with the food donors.

## Tech Stack

**Client:** Angular, HTML, CSS, JavaScript, Bootstrap

**Server:** Node, Express

**DataBase:** MongoDB, PostgreSQL


## Angular Modules Used

 - Angular Material
 - Angular Routes
 - Angular Forms (Template Driven)

## Node Modules Used

- Express
- JWT
- bcrypt
- Mongoose

## Features (UI & Backend)

- Single Page Application (SPA)
- Responsive Design
- Angular Routing
- JSON Web Token Authorization
